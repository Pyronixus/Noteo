Centre d'Aide

🚀 Premiers pas - Créer un compte

Pour commencer :

  1. Cliquez sur "Créer un nouveau compte" 📝
  2. Remplissez votre prénom et nom ✍️
  3. Choisissez semestres ou trimestres 📅
  4. Cliquez sur "Créer un compte" ✅

Astuce : Vos données sont sauvegardées automatiquement ! 💾

📚 Ajouter des matières et notes

Comment ajouter vos notes :

  1. Allez dans l'onglet "Saisie" 📝
  2. Cliquez sur "Ajouter une matière" ➕
  3. Entrez le nom de la matière (ex: Mathématiques) 📖
  4. Collez vos notes depuis Pronote (ex: 15/20 + 12/20) 📋
  5. Cliquez sur "Ajouter" ✅

💡 Utilisez le format : note/dénominateur (ex: 15/20). Ajoutez plusieurs notes avec + !

📊 Analyser vos résultats

Découvrez vos performances :

  1. Cliquez sur l'onglet "Analyses" 📈
  2. Sélectionnez les matières à afficher 📋
  3. Choisissez un type de graphique (barres, lignes, etc.) 📊
  4. Cliquez sur "Mettre à jour" 🔄

🎯 Mode plein écran disponible pour une meilleure vue !

🔒 Sécuriser votre compte

Protégez vos données :

  1. Allez dans les paramètres ⚙️
  2. Cliquez sur "Sécurité" 🔐
  3. Choisissez un code PIN (4, 6 ou 8 chiffres) 🔢
  4. Confirmez votre PIN ✅

🛡️ Votre compte sera verrouillé automatiquement après 15 minutes d'inactivité.

🎨 Personnaliser l'application

Rendez-la à votre goût :

  1. Ouvrez les paramètres ⚙️
  2. Choisissez votre thème (clair, sombre, système) 🌙☀️
  3. Ajustez le volume des sons 🔊
  4. Activez le mode Zen pour une interface épurée 🧘

🎭 Essayez la police dyslexique dans l'accessibilité pour une meilleure lisibilité !

💾 Sauvegarder et restaurer

Protégez vos données :

  1. Allez dans paramètres > Données 📁
  2. Cliquez "Exporter le compte" 💾
  3. Sauvegardez le fichier JSON 📄
  4. Pour restaurer : "Importer un fichier" ⬆️

🔄 Export sélectif par période disponible !

🔄 Changer de compte

Gérer plusieurs comptes :

  1. Cliquez sur votre avatar en haut à droite 👤
  2. Sélectionnez "Changer d'utilisateur" 🔄
  3. Choisissez un compte existant ou créez-en un nouveau ➕

📱 Idéal pour gérer les notes de plusieurs enfants !

❓ Dépannage

Problèmes courants :

  • 🔄 Rechargez la page si les données ne s'affichent pas
  • 🗑️ Videz le cache du navigateur si nécessaire
  • 📱 Assurez-vous d'utiliser un navigateur moderne
  • 💾 Vérifiez que JavaScript est activé

🚨 En cas de problème persistant, contactez le support !

Moyenne générale

Noteo calcule la moyenne générale en faisant la moyenne simple des moyennes de chaque matière, sans tenir compte des coefficients.

Sur Pronote, la moyenne est souvent calculée avec des coefficients (plus ou moins de poids par matière), ce qui peut donner une valeur différente.

Si vous souhaitez que Noteo affiche une moyenne qui correspond à celle de votre établissement, vous pouvez saisir votre moyenne manuellement dans les paramètres.

Politique de confidentialité

Préambule

La présente politique de confidentialité définit et vous informe de la manière dont Noteo utilise et protège les informations que vous nous transmettez, le cas échéant, lorsque vous utilisez la présente application. Noteo s'engage à ce que la collecte et le traitement de vos données soient conformes au Règlement Général sur la Protection des Données (RGPD).

1. Introduction

Bienvenue sur Noteo. Nous nous engageons à protéger vos données personnelles et à assurer la transparence de nos services.

2. Collecte des Données

Nous collectons uniquement les informations nécessaires au bon fonctionnement de l'application : informations de compte (nom, email si code pin configuré), contenu généré par l'utilisateur (notes, documents), données techniques (logs, type d'appareil).

3. Utilisation des Données

Vos données sont utilisées pour stocker vos préférences et vos notes localement sur votre appareil via le localStorage, garantissant un accès rapide et une utilisation hors ligne.

4. Protection des Données

Nous mettons en œuvre des mesures de sécurité robustes pour empêcher tout accès non autorisé, modification ou divulgation de vos informations.

5. Partage avec des Tiers

Noteo ne vend jamais vos données personnelles. Le partage ne s'effectue que dans les cas suivants : conformité légale, prestataires de services tiers (hébergement, analyses) sous contrats stricts de confidentialité.

6. Vos Droits

Vous disposez d'un droit d'accès, de rectification et de suppression de vos données personnelles directement via les paramètres de votre compte ou en nous contactant.

7. Modifications

Nous nous réservons le droit de modifier ces politiques à tout moment. Les utilisateurs seront informés de tout changement majeur via l'application.

Politique de gestion de données

Stockage des Données

Toutes vos données (notes, paramètres, comptes) sont stockées localement dans votre navigateur via le localStorage. Aucune donnée n'est transmise à des serveurs externes.

Sauvegarde et Export

Vous pouvez exporter vos données à tout moment via les paramètres. Les exports sont au format JSON et peuvent être réimportés pour restaurer vos données.

Suppression des Données

Vous pouvez supprimer vos données individuellement ou en totalité via les paramètres. La suppression est immédiate et irréversible.

Sécurité

Les données sont chiffrées localement si vous utilisez un code PIN. Sans PIN, les données sont accessibles à quiconque utilisant votre appareil.

Responsabilité

Vous êtes responsable de la sauvegarde de vos données. Noteo ne peut être tenu responsable de la perte de données due à des problèmes techniques ou à une utilisation incorrecte.
